### Ejecutar CRUD

- `carpeta - rnacademic`

### Verificar ip local y reemplazar en el archivo app.js

- `http://${ip-local}/rnacademic/InsertStudentData.php`
- `http://${ip-local}/rnacademic/ShowAllStudentsList.php`
- `http://${ip-local}/rnacademic/DeleteStudentRecord.php`

- `npm install`
